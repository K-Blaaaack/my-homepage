## 迁移说明

本站已从纯静态 HTML 迁移至 [Hugo](https://gohugo.io/) 静态站点生成器。

### 为什么要迁移？

- 更好的内容管理
- 支持 Markdown 写作
- 自动生成 sitemap
- 更方便的部署流程

### 技术栈

| 组件 | 技术 |
|------|------|
| 构建器 | Hugo |
| 部署 | Cloudflare Workers |
| 样式 | 原生 CSS |
